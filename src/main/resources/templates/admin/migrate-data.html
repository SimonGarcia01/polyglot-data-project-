Migrar Datos de SQL a MongoDB

¿Qué hace esta herramienta?

Esta herramienta migra todos los usuarios desde la base de datos SQL (PostgreSQL/H2) a MongoDB.

Mapea automáticamente:

  • Student → Usuario MongoDB con rol "student"
  • Employee → Usuario MongoDB con rol "collaborator"
  • Trainer/Admin → Usuario MongoDB con rol "trainer"

Opciones de Migración

Migración Incremental

Solo migra usuarios nuevos que aún no existen en MongoDB.

Recomendado si ya tienes datos en MongoDB y solo quieres sincronizar nuevos usuarios.

Migración Completa (Limpiar y Migrar)

⚠️ ADVERTENCIA: Esta opción eliminará TODOS los usuarios existentes en MongoDB y los reemplazará con los datos de SQL.

Usa esta opción solo si quieres sincronizar completamente desde cero.

Notas Importantes

  • La migración solo incluye usuarios activos (isActive = true)
  • Los usuarios deben tener un Student o Employee asociado
  • El campo sql_user_id en MongoDB apunta al ID de Student/Employee en SQL
  • Puedes ejecutar la migración incremental múltiples veces sin problemas
